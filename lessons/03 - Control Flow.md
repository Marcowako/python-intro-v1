# Lesson 3 — Control Flow

**Lesson Overview**

Right now, the programs you've written run the same way every time. This week that changes — you'll learn to write programs that make decisions: responding differently based on what the user types, what values are stored, or what conditions hold true. That's the skill that turns a static script into an interactive program. You'll also set up your GitHub workflow; from here on, all assignments are submitted via pull request.

**Learning Objectives**

This week, I can...

* Write if/elif/else statements that respond to different conditions.
* Combine conditions using and, or, and not, and identify truthy and falsy values.
* Build an interactive, text-based program that uses input() together with control flow.
* Clone a repository, make a commit, and open a pull request on GitHub.

   Applying conditionals and boolean logic to build interactive, text-based programs (e.g., a simple quiz or decision tree); using
