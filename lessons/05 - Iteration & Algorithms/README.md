# Lesson 5 — Iteration & Algorithms

**Lesson Overview**

You already know how to write a loop — this week you'll sharpen that into something more precise. You'll learn the patterns that appear again and again in real code: counting, accumulating, and searching. Then you'll zoom out and start thinking like a computer scientist: what does it actually mean to sort a list efficiently, and how does a computer find something quickly in a large dataset? These are questions programmers have been refining for decades, and understanding them will change how you approach writing code.

**Learning Objectives**

This week, I can...

* Write loops that count occurrences, accumulate totals, and search for values in a sequence.
* Write a list comprehension and explain when to use one instead of a loop.
* Implement a linear search and explain how binary search works conceptually.
* Identify and fix common loop bugs: off-by-one errors, infinite loops, and incorrect accumulator logic.
