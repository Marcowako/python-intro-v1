# Lesson 10 — Final Project Part I

**Lesson Overview** <br>
The final project spans two weeks. This week you build the foundation that every student completes: a Python program that connects to a real public API, transforms the data it returns, and lets a user interact with it through a command-line interface.

Next week you'll extend what you build here by choosing one of two tracks — a data visualization or a data pipeline. Your Week 11 work builds directly on top of Week 10, so getting the core program right this week matters.

Before you start coding, read through the [Final Project Overview](https://github.com/Code-the-Dream-School/python-intro-v1/blob/main/resources/final-project-overview.md) to understand the full two-week arc, the API options, and the rubric your project will be evaluated against.

## Topics

1. **[Planning Your Project](01_planning.md)**
   Phase 0: exploring your API in the browser before writing code; identifying the fields you'll use; sketching your function signatures; setting up the starter repository

2. **[Building the Core Program](02_core_program.md)**
   Phase 1: fetching data with `requests`; parsing the JSON response into a list of dictionaries; wrapping calls in `try/except`; organizing logic into functions with clear parameters and return values

3. **[Building the CLI](03_cli_tool.md)**
   Phase 2: accepting user input with `input()` or `argparse`; implementing at least one meaningful interaction (filter, lookup, or comparison); formatting results for the terminal; handling unexpected input without crashing
