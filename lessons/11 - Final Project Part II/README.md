# Lesson 11 — Final Project Part II

**Lesson Overview**

This is the final week of the course!

Week 11 has two options. Both take what you built in Week 10 and add a meaningful layer on top. Read this page before you start building, then go to your chosen track's lesson and read it before writing any code.

**Learning Objectives**

This week, I can...

* Extend an existing Python project by adding a new feature in my chosen track.
* (Option A) Build a chart from my API data using matplotlib, label it, and save it as a PNG.
* (Option B) Clean fields from my API data and export the results to a CSV using `csv.DictWriter`.

## Topics

Remember that you only need to choose either Option A or Option B.

2. **[Option A — Data Visualization with matplotlib](https://github.com/Code-the-Dream-School/python-intro-v1/blob/main/lessons/11%20-%20Final%20Project%20Part%20II/02_option_a_visualization.md)**
   
   Installing `matplotlib`; building a basic chart from hardcoded data; adapting the chart to your API data; labeling axes and adding a title; saving your chart as a PNG
