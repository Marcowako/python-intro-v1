# Lesson 4 — Data Structures & Loops

**Lesson Overview**

Most real programs don't work with a single value, they work with collections: a list of users, a dictionary of settings, a set of unique tags. This week you'll learn Python's four core data structures (lists, tuples, dictionaries, and sets) and how to loop through them. Those two skills combined put you within reach of a whole new category of programs.

**Learning Objectives**

This week, I can...

* Create and manipulate lists and tuples and explain why mutability matters.
* Create and access dictionaries and sets and choose the right structure for a given problem.
* Write `for` and `while` loops, including using `break` and `continue`.
* Loop over a list of dictionaries to access and display structured data.
