# Lesson 3 — Control Flow

**Lesson Overview**
Back to Python! This week, you will learn to write programs that make decisions using conditionals and boolean logic.You will also set up your GitHub workflow — from here on, all assignments are submitted via pull request.

## Topics

1. **[if/else and Comparison Operators](01_if_else_comparison.md)**
  
   `if`, `elif`, `else`; comparison operators (`==`, `!=`, `<`, `>`, `<=`, `>=`); Python indentation rules and code blocks

2. **[Boolean Operators](02_boolean_operators.md)**
  
   `and`, `or`, `not`; combining conditions; truthy and falsy values

3. **[Decision-Making Scripts](03_decision_scripts.md)**

   Applying conditionals and boolean logic to build interactive, text-based programs (e.g., a simple quiz or decision tree); using `input()` with control flow

4. **[GitHub Workflow: First Commit & PR](04_github_workflow.md)**
 
   Cloning a repository locally; `git add`, `git commit`, `git push`; opening a pull request on GitHub; the assignment submission workflow used for the rest of the course
