# Lesson 6 — Functions & Scope

**Lesson Overview**

Copy-pasting code to reuse it works, until it doesn't. The moment you need to change the logic, you have to change it everywhere it appears, and one missed spot is a bug. This week you'll learn the better approach: functions. You'll define them, pass data in and out, understand where variables live, and then put that knowledge into practice by refactoring an earlier script. You can see firsthand what cleaner, more modular code actually looks like.

**Learning Objectives**

This week, I can...

* Define a function with parameters, default values, and a return value.
* Explain the difference between local and global scope and identify scope-related bugs.
* Refactor a script into named functions and explain why the result is easier to read and reuse.
